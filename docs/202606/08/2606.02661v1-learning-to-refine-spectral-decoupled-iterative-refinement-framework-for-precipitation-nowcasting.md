## 摘要
准确的降水临近预报对于减灾至关重要，但深度学习方法面临一个关键权衡：回归模型产生过度平滑、频谱衰减的预测，模糊了对流细节并违反湍流幂律；扩散模型生成逼真但缺乏物理基础的随意幻觉。我们提出了频谱解耦迭代细化（SDIR），这是一个确定性框架，将临近预报重新表述为渐进式频率解耦细化。SDIR首先提取稳定的低频天气尺度骨架，然后在物理约束下迭代细化高频纹理，消除了模糊和幻觉。它采用双路径设计：具有尺度自适应Transformer的天气尺度频率引导器（SFG-Former）用于全局结构，以及具有尺度条件傅里叶神经算子的傅里叶残差细化器（FR-Refiner）用于精细残差。具有动态掩蔽的物理一致功率谱密度（PCPSD）损失强制实现湍流一致的频谱分布。在三个基准上的实验表明，SDIR在空间准确性上显著优于现有最先进方法，同时实现了与基于扩散方法相媲美的频谱保真度，从而实现了可靠的高分辨率业务临近预报。代码链接：https://github.com/RuntimeWarning/SDIR。

## Abstract
Accurate precipitation nowcasting is vital for disaster mitigation, but deep learning methods face a key trade-off: regression models produce over-smoothed, spectrally decaying predictions that blur convective details and violate turbulence power laws; diffusion models generate realistic yet unanchored hallucinations lacking physical grounding. We propose Spectral-Decoupled Iterative Refinement (SDIR), a deterministic framework that reformulates nowcasting as progressive frequency-decoupled refinement. SDIR first extracts a stable low-frequency synoptic skeleton, then iteratively refines high-frequency textures under physical constraints, eliminating both blurring and hallucinations. It features a dual-path design: the Synoptic Frequency-Guided Former (SFG-Former) with Scale-Adaptive Transformers for global structure, and the Fourier Residual Refiner (FR-Refiner) with Scale-Conditioned Fourier Neural Operators for fine residuals. A Physically Consistent Power Spectral Density (PCPSD) loss with dynamic masking enforces a turbulence-consistent spectral distribution. Experiments on three benchmarks show SDIR significantly outperforms SOTA methods in spatial accuracy while achieving spectral fidelity competitive with diffusion-based methods, enabling reliable high-resolution operational nowcasting. Code link: https://github.com/RuntimeWarning/SDIR.

- **整体意义**：论文旨在弥合上述权衡，提出一个既能保持结构性可靠（如回归模型），又能恢复精细光谱细节（如生成模型）的**确定性**框架，以实现物理上合理、高分辨率的业务化临近预报。

### 2. 论文提出的方法论 (核心思想、关键技术、流程)

- **核心思想**：将降水临近预报重新定义为一个**确定性的、由粗到细的频谱解耦迭代细化过程**。该过程受大气动力学中能量串级现象的启发：大尺度天气系统（低频）的演变是稳定的，为小尺度对流细节（高频）提供边界条件。
- **关键技术**：
    1.  **频谱感知训练课程 (Spectral-Aware Training Curriculum)**：引入频率尺度信号 `s`。通过离散余弦变换截断，从真实标签 `Y` 中提取不同带宽的低频条件 `C_s`。训练时从偏向低频的Beta分布中采样 `s`，迫使模型先学会粗结构，再处理精细纹理。
    2.  **双路径混合架构**：
        - **天气尺度频率引导器 (SFG-Former)**：负责提取稳定的**低频天气尺度骨架**。使用尺度自适应Transformer和3D旋转位置编码，捕捉长程时空依赖关系，生成结构基础 `Y_base`。
        - **傅里叶残差细化器 (FR-Refiner)**：负责合成被抑制的**高频残差纹理**。采用基于傅里叶神经算子的U-Net结构，在频域进行全局操作，高效生成高频残差 `Y_res`。最终的预测 `Y = Y_base + Y_res`。
    3.  **物理一致功率谱密度损失 (PCPSD Loss)**：在频域监督预测结果，使其能量分布符合大气湍流功率谱定律。该损失函数包含一个动态掩蔽机制 `Ω(k, s)`，根据当前迭代步 `s` 动态调整对不同频段的监督权重，对尚未解锁的高频段施加更强的约束。
    4.  **频率解锁推理 (Frequency-Unlocking Inference)**：推理阶段，模型从“冷启动”（s=0）开始，按预设计划逐步增大 `s`，反复执行“预测-提取低频条件-再预测”的迭代过程，渐进式地生成高分辨率预测。

- **算法流程**：
    - **训练**：
        1.  采样频率尺度 `s`。
        2.  通过DCT截断得到低频条件 `C_s`。
        3.  SFG-Former根据历史输入X和条件 `C_s` 预测结构骨架 `Y_base`。
        4.  FR-Refiner根据历史X、骨架 `Y_base` 和条件 `s` 预测残差 `Y_res`。
        5.  最终预测 `Y_hat = Y_base + Y_res`。
        6.  使用空间损失（`L1`）和PCPSD损失联合优化。
    - **推理**：
        1.  初始化 `s=0`。
        2.  循环 `K` 步（如8步），每步按递增计划使用当前的 `s` 进行预测。
        3.  对于非最后一步，根据当前预测提取新的、更高频率的 `C_s` 作为下一步的输入。
